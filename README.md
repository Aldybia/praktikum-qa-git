# 🛡️ Praktikum Quality Assurance (Manual & Automation)

Proyek ini merupakan portofolio pengujian perangkat lunak yang saya kerjakan dalam program **National Digital Talent Academy**. Fokus utama adalah implementasi pengujian pada alur Login menggunakan pendekatan manual dan otomatis.

## 📋 Cakupan Pengujian
1. **Manual Testing:** Perencanaan skenario uji melalui [Test Case Dokumentasi](docs/test-cases.md).
2. **Automation Testing:** Pembuatan skrip uji menggunakan Python & Pytest.
3. **CI/CD Integration:** Otomatisasi eksekusi tes menggunakan **GitHub Actions**.

## 🛠️ Tech Stack
- **Languages:** Python
- **Testing Tools:** Pytest, Postman (API)
- **CI/CD:** GitHub Actions
- **Documentation:** Google Sheets
