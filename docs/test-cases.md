# Dokumentasi Pengujian Manual - Fitur Login

Daftar skenario ini digunakan untuk memastikan fungsi autentikasi berjalan dengan benar sebelum masuk ke tahap otomatisasi.

| ID | Skenario Pengujian | Langkah-langkah | Hasil yang Diharapkan | Status |
| :--- | :--- | :--- | :--- | :--- |
| TC01 | Login Berhasil | Input user & pass valid -> Klik Login | Masuk ke Dashboard | Passed |
| TC02 | Login Gagal (Pass Salah) | Input user valid & pass salah -> Klik Login | Pesan error muncul | Passed |
| TC03 | Form Kosong | Klik Login tanpa isi data | Pesan "Field Required" | Passed |
